## ROLE

You are building **slabsvinyl.com** — a curated archive of new-condition, factory-sealed vinyl records. The aesthetic is archival, precise, and warm. Think industrial parts catalog meets record store liner notes. Every design decision should feel like it was made by a human who deeply understands both graphic design and vinyl culture — not generated by AI.

---

## BLOCK 1 — TAILWIND CONFIG

Use Tailwind CSS v3. All tokens are defined in `/tailwind.config.js`. Do not use Tailwind defaults for colors, fonts, or border-radius.

---

## BLOCK 2 — DESIGN SYSTEM DIRECTIVES

### Font Zone Rules — Non-Negotiable

| Font | Role | Size Range |
|---|---|---|
| Instrument Serif | Headings, hero, editorial | 24px → 128px |
| Funnel Display | Nav, labels, overlines, archival metadata | 11px → 20px |
| Funnel Sans | Body copy, product data, form fields | 12px → 18px |

### Color Rules
- **Amber (`#CD9121`) BANNED as text on light backgrounds** — use on dark surfaces only
- **Prices in light mode:** Forest (`#204F41`)
- **Prices in dark mode:** dark-accent (`#E8A830`)
- Page background: Paper (`#F6F6F0`) — never `#FFFFFF`

### Border Radius — Zero Everywhere
`border-radius: 0` on ALL components. Only exception: vinyl disc (`border-radius: 50%`).

### Action Hierarchy
| Level | Visual | Use |
|---|---|---|
| Primary | Forest bg + Cream text, full width | "Grab it" — PDP only |
| Secondary / Ghost | 1px border, no fill | Secondary PDP actions |
| Tertiary | Text + `→`, no border | "View all →" |
| Quaternary | Underline only | Inline editorial links |

### Motion
All transitions: `cubic-bezier(0.25, 0.1, 0.25, 1)`. Never bouncy or springy.

---

## BLOCK 3 — HARD CONSTRAINTS

### Never Do
- `border-radius` on anything except vinyl disc
- Amber as text on light surfaces
- Buy buttons in product grid
- Pill-shaped badges
- `font-family: Inter`, `Roboto`, `system-ui`
- White (`#FFFFFF`) backgrounds
- Tailwind default color names (`blue-500`, `gray-200`, etc.)

### Product Card
- No buy button in grid — lives on PDP only
- Album cover links to PDP — only interactive element
- Vinyl disc slides RIGHT on hover: `translateX(0%)` → `translateX(55%)`
- Parent must use `overflow: visible`
